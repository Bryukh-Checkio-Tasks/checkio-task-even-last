Mit Abschnitten und der built-in Funktion "sum" kannst Du diese Aufgabe lösen. Abschnitte bestehen aus drei Angaben - "from", "to" und "step". Wenn du hirfür keine Werte angibst, werden die Standardwerte genutzt.
Um also alle geraden Indizes zu erhalten nutze: array[::2], das ist eine kürzere Variante als: array[0:len(array):2]

If you don't want to use slices and sum, just accumulate the sum in a variable. 
def checkio(array):
    result = 0
    for i in range(0, len(array), 2):
        result += array[i]  # result = result + array[i]
