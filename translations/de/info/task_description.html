Vorgegeben wird ein Array mit Integern. Du sollst die Summe der Elemente mit geraden Indizes (0te, 2te,4te...) ermitteln und diese Summe mit dem letzten Element des Array multiplizieren.
Vergiss nicht, dass das erste Element eines Arrays den Index 0 hat.

Bei einem leeren Array, wird das Ergebnis immer 0 (Null).

Eingabe: Eine Liste mit Integern.

Ausgabe: Die Zahl als Integer.

Beispiele: 

checkio([0, 1, 2, 3, 4, 5]) == 30
checkio([1, 3, 5]) == 30
checkio([6]) == 36
checkio([]) == 0

Wozu das gut ist: Indizes und Abschnitte sind wichtige Elemente beim programmieren in Python und anderen Programmiersprachen. Diese Aufgabe wird sich später als nützlich erweisen.

Voraussetzung: 0 ≤ len(array) ≤ 20
all(isinstance(x, int) for x in array)
all(-100 < x < 100 for x in array)
