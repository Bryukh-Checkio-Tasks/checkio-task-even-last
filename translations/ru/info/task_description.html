Дан массив целых чисел. Нужно найти сумму элементов с четными индексами (0, 2, 4 и т.д.), затем перемножить эту сумму и последний элемент исходного массива. Помните, что первый элемент массива имеет индекс 0.

Для пустого массива результат всегда 0 (ноль).

На входе: Список (list) целых чисел (int).

На выходе: Целое число (int).

Примеры: 

checkio([0, 1, 2, 3, 4, 5]) == 30
