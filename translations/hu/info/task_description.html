Adott egy egész számokból álló tömb. Meg kell találni a tömb páros (a 0-ik, a 2-ik, a 4-ik, stb.) pozícióiban álló elemeinek az összegét, majd megszorozni ezt az összeget a tömb legutolsó elemére. Ne feledje, hogy a tömb első elemének indexe 0!

Az üres tömb számára az eredmény mindig 0 (nulla).

Bemeneti adatok: egy egész számokból (int) álló lista (list).

Kimeneti adatok: egy egész szám (int).

Példák: 

{% if interpreter.slug == "js-node" %}
evenLast([0, 1, 2, 3, 4, 5]) == 30
evenLast([1, 3, 5]) == 30
evenLast([6]) == 36
evenLast([]) == 0
{% else %}
checkio([0, 1, 2, 3, 4, 5]) == 30
checkio([1, 3, 5]) == 30
checkio([6]) == 36
checkio([]) == 0
{% endif %}

Alkalmazása: Az indexek és a tömbszeletek a programozás nagyon fontos részei mind a Pythonon, mind más programozási nyelveken. A továbbiakban jó hasznát lehet venni ezeknek.

Előfeltételek: 0 ≤ len(array) ≤ 20
all(isinstance(x, int) for x in array)
all(-100 < x < 100 for x in array)
