あなたは整数の配列が与えられます。 あなたはインデックスが偶数(0番目、2番目、4番目…)である要素の和を求めてから、この和と配列の最後の要素を掛けなければいけません。 最初の要素のインデックスが0であることを忘れてはいけません。

空の配列については、結果は常に0(ゼロ)になるでしょう。

入力: 整数のリスト

出力: 結果の数値、整数

例: 

{% if interpreter.slug == "js-node" %}
evenLast([0, 1, 2, 3, 4, 5]) == 30
evenLast([1, 3, 5]) == 30
evenLast([6]) == 36
evenLast([]) == 0
{% else %}
checkio([0, 1, 2, 3, 4, 5]) == 30
checkio([1, 3, 5]) == 30
checkio([6]) == 36
checkio([]) == 0
{% endif %}

どのように使われるか: インデックスとスライスはPythonと他の言語で重要な要素です。 これは今後役に立つことでしょう!

事前条件: 0 ≤ len(array) ≤ 20
all(isinstance(x, int) for x in array)
all(-100 < x < 100 for x in array)
